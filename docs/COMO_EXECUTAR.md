# COMO_EXECUTAR.md - Guia de Setup Local

Este guia é para a equipe de desenvolvimento que precisa rodar o sistema localmente.

## Pré-requisitos
* **ASP.NET SDK 6.0 ou superior**
* **SQL Server** (Para o banco de dados)

## 1. Configuração

1.  Clone o projeto.
2.  Configure a string de conexão do banco de dados no arquivo de ambiente.

## 2. Iniciar o Projeto

1.  Abra o terminal na pasta do projeto Backend.
2.  Rode o comando para iniciar: `dotnet run`
