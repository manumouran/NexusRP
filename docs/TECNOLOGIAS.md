# TECNOLOGIAS.md - NexusRP

Este documento detalha o conjunto de tecnologias (Tech Stack) selecionado para o desenvolvimento do NexusRP.

## Backend e APIs
* **C# (ASP.NET Core):** Usado para a lógica de negócio e APIs.
* **Discord API:** Usado para integração em tempo real (denúncias automáticas).

## Banco de Dados
* **MS SQL Server:** Armazenamento seguro de dados de players e punições.
* **Entity Framework Core:** Ferramenta para gerenciar o banco de dados via código C#.

## Frontend
* **HTML, CSS, JavaScript:** Interface administrativa (Dashboard) para a Staff.
